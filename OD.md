### Supported
* OLAINDEX
* Directory Lister
* AutoIndex PHP
* Abyss Web Server X1 & X2
* LiteSpeed Web Server
* Apache
* NGINX
* Evoluted Directory Listing Script
* lighttpd
* phpBB
* OneManager
* h5ai
* Microsoft-IIS
* snif
* OdIndex

**May also work with other ODs. So long as they have `<a>`.*

### Unsupported
* ZFile
* and other JavaScript only ODs
